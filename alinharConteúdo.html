Alinhamento

Posicione os itens do elemento flexível com espaço entre as linhas

center -- Os itens são posicionados no centro do contêiner

flex-start -- Os itens são posicionados no início do contêiner

flex-end -- Os itens são posicionados no final do contêiner

space-between -- Os itens são posicionados com espaço entre as linhas

space-around -- Os itens são posicionados com espaço antes, entre e depois das linhas

initial --Define esta propriedade para seu valor padrão.

inherit -- Herda esta propriedade de seu elemento pai.
